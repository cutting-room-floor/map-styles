AFGHANISTAN & PAKISTAN

== PREREQUISITES ==========================================

While you may modify these styles to work with any data, to use them as
they are you will need to download the following third-party resources.

---- Shapefiles -------------------------------------------

Afghanistan OpenStreetMap data
http://downloads.cloudmade.com/asia/afghanistan/afghanistan.shapefiles.zip

Afghanistan-Pakistan border
http://finder.geocommons.com/overlays/12047.zip

Pakistan OpenStreetMap data
http://downloads.cloudmade.com/asia/pakistan/pakistan.shapefiles.zip

---- Fonts ------------------------------------------------

Aller Regular
http://www.fontsquirrel.com/fonts/Aller
